Xylume TestNet uses a Directed Acyclic Graph (DAG) for parallel, secure transaction validation. Parent-child transaction model ensures efficient processing.

PoUTX

Proof of Unspent Transaction eXploration verifies transactions using unspent outputs, prevents double-spending, and enables lightweight state dependencies.

Attack Resistance

Security
